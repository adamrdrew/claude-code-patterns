# Available Skills

Any of these skills can be invoked by name via the Skill tool

---

```
name: skill-compete
description: Run two competing skill versions on the same input and capture their outputs for comparison
```
---
```
name: fitness-eval
description: Evaluate and compare skill outputs using fitness criteria to determine which version is superior
```
---
```
name: skill-promote
description: Promote a winning skill version to champion status, archiving the previous champion
```
